# Personal Portfolio

My personal portfolio showcasing selected projects and technical work. Built with modern web technologies for performance, clarity, and visual impact.

---

## Overview

This is a responsive, dark-themed portfolio designed to highlight projects and skills with subtle motion and interactive elements.

**Sections include:**

* **Hero** — introduction, personal statement, and links
* **Projects** — selected work with descriptions and visuals
* **Skills** — categorized tech stack
* **Contact** — direct links for collaboration and outreach

---

## Tech Stack

* **Framework:** Next.js
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **3D / Graphics:** Three.js, React Three Fiber
* **Animation:** Framer Motion

---

## Getting Started

```bash
git clone https://github.com/nebyathhailu/personal-portfolio.git
cd personal-portfolio
npm install
npm run dev
```

Open http://localhost:3000 in your browser.
